# ConectaHub

Aplicativo Android desenvolvido com o objetivo de praticar e aprimorar conhecimentos em desenvolvimento mobile.

## Sobre o Projeto

O ConectaHub é um aplicativo criado como parte do meu processo de aprendizado na área de Tecnologia da Informação, com foco em desenvolvimento Android.

O projeto foi desenvolvido aplicando conceitos de estruturação de projeto, organização de arquivos, permissões e geração de APK para instalação.

## Tecnologias Utilizadas

- Java / Kotlin (ajuste para o que você usou)
- Android Studio
- Android SDK
- Gradle

## Funcionalidades

- Estrutura completa de aplicativo Android
- Configuração de permissões no AndroidManifest
- Organização de Activities
- Geração de APK para instalação
- Interface funcional e compilada com sucesso

## Objetivo

Este projeto foi desenvolvido com o objetivo de:

- Praticar desenvolvimento mobile
- Compreender o processo de build e geração de APK
- Evoluir na criação de aplicativos Android
- Construir portfólio para oportunidades na área de T.I

## Status do Projeto

 Em desenvolvimento – melhorias e novas funcionalidades serão adicionadas em breve.
